# 個別プロトコル調査（中プロンプト）

**変更箇所のみを対象とした軽量検証。デプロイ前の最終確認に使用。**

---

## 全体プロトコル調査との違い

| 項目 | 全体 | 個別 |
|------|------|------|
| 対象 | 全コード | 変更箇所のみ |
| スクショ | あり | なし |
| UXチェック | あり | なし |
| 所要時間 | 30分〜 | 5分以内 |

※ プロトコル準拠チェックの内容は同一

---

## 実行順序

```
1_scope.md    → 変更範囲特定（git diff）
2_protocol.md → プロトコル準拠チェック
3_report.md   → 結果レポート出力
```

---

```bash
afplay /System/Library/Sounds/Glass.aiff
```

## 禁止

- コード変更
- DB変更
- ファイル修正
- 問題の自己修正

## やること

- 観察
- 分析
- 問題発見 → 修正依頼を出す
